# 🤖 AI Email Scheduling Assistant

An intelligent AI-powered assistant that automates email-based meeting scheduling and coordination.

---

## 🚀 Features

- 📧 Parses email conversations to detect availability  
- 🤝 Coordinates meeting schedules between multiple participants  
- ⏰ Suggests optimal meeting times automatically  
- 🧠 Uses AI to understand natural language in emails  
- 🔄 Reduces manual back-and-forth communication  

---

## 🛠️ Tech Stack

- Python  
- AI/ML (NLP-based processing)  
- Google APIs (for email integration)  
- Cursor IDE
